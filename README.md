# React-Express-CarRental

## 📋 Description
Le projet de gestion de location de véhicules pour l’agence **"Two Go Car Rental"** a pour objectif principal l’automatisation et la numérisation complète du processus de réservation. 

En réponse aux défis posés par le système traditionnel, ce système a été conçu pour garantir une expérience utilisateur fluide et une gestion administrative efficace, permettant de passer d'une gestion manuelle à une plateforme numérique intégrée.

## 🚀 Fonctionnalités Principales (Besoins Fonctionnels)
* **Gestion des Véhicules :** Ajout, modification et suppression des voitures disponibles dans le parc.
* **Système de Réservation :** Réservation en ligne avec suivi de l'état des demandes.
* **Authentification :** Gestion sécurisée des comptes utilisateurs et des profils.
* **Tableau de Bord Admin :** Interface pour la gestion des locations, des clients et des statistiques.

## 🛠️ Stack Technique
Ce projet est développé en utilisant la stack **MERN** (React/Node) :
* **Frontend :** React.js
* **Backend :** Node.js / Express.js
* **Base de données :** MongoDB
* **Style :** CSS / Bootstrap

## ⚙️ Installation et Configuration
Pour lancer ce projet localement, suivez ces étapes :

### 1. Cloner le projet
```bash
git clone [https://github.com/LAMGAMATE/React-Express-CarRental.git](https://github.com/LAMGAMATE/React-Express-CarRental.git)
cd React-Express-CarRental

2. Configuration du Backend (Serveur)
Bash
cd server
npm install
npm start
3. Configuration du Frontend (Client)
Ouvrez un nouveau terminal et exécutez :

Bash
cd client
npm install
npm start
